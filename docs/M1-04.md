# M1-04 设计说明（src/managed 解决方案与项目占位）

1. 在仓库根目录创建 `XServerByAI.Managed.sln`，作为 `src/managed` 下 C# 项目的统一入口。
2. 在 `src/managed/Foundation` 创建类库项目，占位公共类型与后续互操作共享契约。
3. 在 `src/managed/GameLogic` 创建类库项目，占位后续业务逻辑程序集。
4. 将 `Foundation` 与 `GameLogic` 注册到解决方案中，确保后续可统一还原、构建与测试。
5. 为 `GameLogic` 添加对 `Foundation` 的项目引用，提前建立后续业务层依赖方向。
6. 提供最小占位类型，保证解决方案可以通过 `dotnet build` 完成编译验证。
